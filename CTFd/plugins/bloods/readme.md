# overview
this repo contains a bloods plugin for the ctfd platform. the plugin adds the popular "first blood" achievements to users and teams.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_bloods" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_bloods"`)
